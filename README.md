# Library Management Database System

## Project Description

This project is a database management system developed for managing a college library.

## Features

- Member Management
- Book Management
- Category Management
- Book Issue
- Book Return
- Fine Management
- Reservation System
- Review System
- Report Generation

## Technologies

- MySQL
- SQL
- GitHub
- VS Code
